📊FULL ADVANCED METRICS TABLE
TEAM FPV (ETxP) FPCR NET ERR DIFF REPEAT SETS RED ZONE PTBs RUCK SPEED SET CTRL PRESS ACC DEF CONVERSION
FPV — Field Position Value (ETxP). Expected points from field position gained, accounting for field zone, tackle number and game state.
FPCR — Field Position Conversion Rate. Actual points scored ÷ FPV. >2.5 = exceptional; <0.8 = poor conversion.
Net Err Diff — Opponent errors minus own errors. Positive = better ball control than opposition.
Red Zone PTBs — Play-the-balls inside opponent's 20m. Highest-value attacking territory.
Ruck Speed — Avg seconds from tackle to play-the-ball. Lower = faster ruck. Strongest predictor of match outcome.
Set Ctrl Score — Composite: completion %, error count, repeat sets earned. Weighted for field position.
Press Acc — Pressure Accumulation Score. Repeat sets earned × 2 + penalties awarded. Measures sustained pressure generation.
Def Conversion — Defensive efficiency: FPV conceded ÷ points conceded. Higher = better defence under pressure.
